2. Template d'Inventaire de Sortie (Exit)

This template is used for creating exit inventory documents when a tenant moves out, including comparison with entry state.

Available Variables:
{{reference}} {{date}} {{adresse}} {{appartement}} {{locataire_nom}} {{equipements}} {{comparaison}} {{observations}}

Features:

  • Professional header with company branding
  • Property and tenant information section
  • Equipment list with current state
  • Comparison section - highlights differences from entry inventory
  • General observations area
  • Signature section for landlord and tenant
  • Clean, modern design with red accents (indicating exit)
  • Alert boxes for warnings and important information

How to Use:

  1. Access the admin panel at /admin-v2/inventaire-configuration.php
  2. The templates will be loaded in the TinyMCE editor
  3. You can customize the HTML and CSS as needed
  4. Click on variable tags to copy them to clipboard
  5. Use the preview function to see how your changes look
  6. Save your changes to update the templates in the database
