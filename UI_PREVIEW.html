UI Preview - Nouvelles Fonctionnalités

Gestion des délais flexibles et tâches automatisées

1. Configuration du Délai Flexible
Nouvelle fonctionnalité: Choisissez l'unité de temps (minutes/heures/jours) pour le délai de réponse automatique.

Délai avant l'envoi automatique de la réponse aux candidatures

Configuration actuelle: 4 jours (ouvrés)
Exemples de configuration:
  • 120 minutes = 2 heures après la soumission
  • 48 heures = 2 jours (calendaires)
  • 4 jours = 4 jours ouvrés (du lundi au vendredi)
2. Gestion des Tâches Automatisées
Nouvelle page: Accédez via le menu Tâches Automatisées pour gérer les cron jobs.
Traitement des candidatures Actif Succès

Traite automatiquement les candidatures et envoie les réponses d'acceptation ou de refus selon les critères configurés

Fichier cron/process-candidatures.php
Fréquence Quotidien
0 9 * * *
Dernière exécution 30/01/2026 09:00:15
[2026-01-30 09:00:12] === Starting candidature processing === [2026-01-30 09:00:12] Using automatic response delay: 4 jours [2026-01-30 09:00:12] Found 3 applications to process [2026-01-30 09:00:13] Processing application #15 for John Doe [2026-01-30 09:00:14] Acceptance email sent to john.doe@example.com [2026-01-30 09:00:14] === Processing complete ===
Configuration serveur requise:

Pour que les tâches s'exécutent automatiquement, configurez le cron sur votre serveur:

# Éditer le crontab crontab -e # Ajouter cette ligne pour exécution quotidienne à 9h00 0 9 * * * /usr/bin/php /path/to/contrat-de-bail/cron/process-candidatures.php
